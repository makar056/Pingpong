## 🛠 Установка

```bash
# Клонировать репозиторий
git clone https://github.com/yourname/ping-pong.git

# Установить зависимости
pip install -r requirements.txt

# Запустить игру
python main.py

```
🕹️ Управление
Команда	Действие
← / →	Движение платформы
Space	Пауза / Продолжить
ESC	Выход в меню
M	Вкл/Выкл музыку
F	Полноэкранный режим
